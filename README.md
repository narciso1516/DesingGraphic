# Marketing Advisor Extension — QA Knowledge Hub

Centro de conocimiento visual para organizar conceptos, flujos y material de QA relacionado con Marketing Advisor Extension.

## Propósito

Facilitar la consulta y el aprendizaje de información de producto y calidad mediante una experiencia web interactiva.

## Tecnología

- HTML
- CSS
- JavaScript
- Diseño de información
- UX/UI

## Uso

Abre `index.html` en un navegador.
